# 狼人杀法官助手

这是一个纯静态网页版本的线下狼人杀法官助手，适合直接发布到 GitHub Pages。

## 功能

- 登记玩家人数、玩家名称和真实身份
- 使用 9 人、10 人、12 人常用板子，或手动调整身份池
- 追加白痴、白狼王、狼美人、骑士、摄梦人、丘比特等备用身份
- 随机分配身份
- 按本局角色生成夜晚法官话术
- 支持编辑当前夜晚步骤的话术
- 记录狼人刀人、预言家查验、女巫用药、守卫守护、猎人状态
- 记录每一夜结算和手动出局日志
- 提示板记录真实身份、存活状态和一次性技能是否用掉
- 手动标记出局时可选择第几夜/第几天和出局方式
- 天亮自动结算平安夜或出局玩家
- 使用浏览器本地存储保存当前局

## 本地打开

直接打开 `index.html` 即可使用。

## 发布到 GitHub Pages

1. 把 `HTML` 文件夹提交到 GitHub 仓库。
2. 在仓库设置里进入 `Pages`。
3. Source 选择当前分支。
4. Folder 如果可以选择子目录，选择 `/HTML`；如果界面只支持根目录或 `/docs`，把 `HTML` 里的文件移动到仓库根目录或 `docs` 目录。
5. 保存后等待 GitHub Pages 生成链接。

## 规则来源

夜晚流程参考 `StoneMoe/wolf` 的基础顺序：

```text
天黑请闭眼 -> 狼人 -> 预言家 -> 女巫 -> 守卫 -> 猎人 -> 结算夜晚死亡 -> 天亮
```

当前网页按线下面杀法官助手处理，重点是提示、记录和结算；白天技能以状态记录和手动出局为主。
